# ToDo List

A simple C++ console application for managing a to-do list.

## Features
- Add new tasks
- Remove tasks by ID
- Mark tasks as completed
- Display task list

## Requirements
- C++20 or newer
- Standard C++ library

## How to Compile and Run

### Using g++
```bash
g++ -std=c++20 main.cpp -o todo_list
./todo_list
